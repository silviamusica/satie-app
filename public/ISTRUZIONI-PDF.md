# Come aggiungere lo spartito PDF

Per visualizzare lo spartito del primo movimento nel lettore PDF:

1. **Trova o scarica** il file PDF dello spartito del primo movimento del Concerto Op. 37 di Beethoven

2. **Rinomina** il file in: `spartito-primo-movimento.pdf`

3. **Copia** il file nella cartella `public/` di questo progetto:
   ```
   beethoven-app/public/spartito-primo-movimento.pdf
   ```

4. Il lettore PDF apparirà automaticamente sotto la sezione analisi del primo movimento

## Fonti per lo spartito

Puoi scaricare lo spartito gratuitamente da:

- **IMSLP (Petrucci Music Library)**: https://imslp.org/wiki/Piano_Concerto_No.3,_Op.37_(Beethoven,_Ludwig_van)
- **MuseScore**: Cerca "Beethoven Piano Concerto 3"

## Formato consigliato

- Formato: PDF
- Dimensione massima consigliata: 50 MB per prestazioni ottimali
- Se il file è troppo grande, considera di estrarre solo le pagine del primo movimento

## Note tecniche

Il lettore PDF supporta:
- ✅ Navigazione tra le pagine (voltapagine)
- ✅ Zoom in/out
- ✅ Visualizzazione ottimizzata per desktop e tablet
- ✅ Caricamento progressivo

Se il file non viene trovato, apparirà un messaggio di avviso con le istruzioni.
